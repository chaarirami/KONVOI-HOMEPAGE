Run the following install command:
```
pnpm add @astrojs/preact preact uplot
pnpm add -D @types/uplot
```

This adds:
- `@astrojs/preact` — Astro integration for Preact islands (devDependency install trigger, but declared in dependencies since Preact islands run in the browser)
- `preact` — the client-side UI framework
- `uplot` — lightweight 12 KB time-series chart library for SensorDataViz

Then edit `astro.config.ts`:

1. Add import at top (after existing imports, before astrowind import):
```typescript
import preact from '@astrojs/preact';
```

2. Add `preact()` to the `integrations` array, BEFORE `compress()` to ensure Preact islands are processed before compression:
```typescript
integrations: [
  sitemap(),
  mdx(),
  preact({ compat: false }),
  icon({ ... }),
  compress({ ... }),
  astrowind({ ... }),
],
```

The `compat: false` flag is correct for new projects using Preact 10 hooks API only (no React compat layer needed — SensorDataViz uses `preact/hooks` directly).

Do NOT install `@types/preact` — Preact 10 ships its own types.
