**Step 1 — Download favicon assets from the live site:**

Run the following commands to download the three favicon files directly from konvoi.eu:


If any URL returns a 404 or HTML error page (check with `file src/assets/favicons/favicon.svg` — it should say "SVG Scalable Vector Graphics image", not "HTML document"), try these fallback paths:
- `https://www.konvoi.eu/icons/favicon.svg`
- `https://www.konvoi.eu/assets/favicon.svg`
- `https://www.konvoi.eu/static/favicon.svg`

If download fails for any file, note it in the task output — the checkpoint will present options to the user.

**Step 2 — Sanitize the SVG favicon:**

SVGs from external sites may contain JavaScript event handlers or `<script>` tags that constitute an XSS risk when served as a page resource (ASVS V12 — Files & Resources).

Run the following grep to detect executable content:
```bash
grep -E "<script|onerror|onload|onclick|javascript:" src/assets/favicons/favicon.svg
```

If any match is found, remove those elements from the SVG manually or use svgo:
```bash
pnpm dlx svgo --config='{"plugins":["removeScriptElement","removeEventListeners"]}' src/assets/favicons/favicon.svg -o src/assets/favicons/favicon.svg
```

If no match is found, the SVG is clean — proceed.

**Step 3 — Update Favicons.astro mask-icon color:**

In `src/components/Favicons.astro`, update the `color` attribute on the `<link rel="mask-icon">` tag from AstroWind purple to Konvoi brand blue:

Change:
```astro
<link rel="mask-icon" href={favIconSvg.src} color="#8D46E7" />
```

To:
```astro
<link rel="mask-icon" href={favIconSvg.src} color="#4F8ED5" />
```

(#4F8ED5 is the hex approximation of `hsl(214.48 38.33% 55.49%)` — Konvoi primary blue per D-02)

The import statements and other link tags in Favicons.astro remain unchanged.
