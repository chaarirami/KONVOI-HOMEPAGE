**Zod schema:**

```typescript
const buildConsultSchema = () => z.object({
  name:          z.string().min(1, 'form.error_required'),
  email:         z.string().min(1, 'form.error_required').email('form.error_email'),
  phone:         z.string().optional(),
  company:       z.string().min(1, 'form.error_required'),
  fleet_size:    z.coerce.number().min(1, 'form.error_fleet_min'),
  message:       z.string().optional(),
  dsgvo_consent: z.literal(true, { errorMap: () => ({ message: 'form.error_dsgvo' }) }),
});
```

**State:**
```typescript
const [fields, setFields] = useState({
  name: '', email: '', phone: '', company: '',
  fleet_size: '', message: '', dsgvo_consent: false,
});
const [errors, setErrors] = useState<Record<string, string>>({});
const [isSubmitting, setIsSubmitting] = useState(false);
const [submitError, setSubmitError] = useState('');
```

**Pre-fill on mount (useEffect):**

**handleChange:**
```typescript
const handleChange = (e: Event) => {
  const target = e.target as HTMLInputElement | HTMLTextAreaElement | HTMLSelectElement;
  const value = target.type === 'checkbox' ? (target as HTMLInputElement).checked : target.value;
  setFields(prev => ({ ...prev, [target.name]: value }));
  // Clear error for this field on edit
  if (errors[target.name]) {
    setErrors(prev => { const n = { ...prev }; delete n[target.name]; return n; });
  }
};
```

**handleSubmit:**
```typescript
const handleSubmit = async (e: Event) => {
  e.preventDefault();
  setIsSubmitting(true);
  setSubmitError('');
  setErrors({});

  // 1. Zod validation
  const schema = buildConsultSchema();
  const result = schema.safeParse({
    ...fields,
    fleet_size: fields.fleet_size === '' ? undefined : fields.fleet_size,
    dsgvo_consent: fields.dsgvo_consent,
  });
  if (!result.success) {
    const newErrors: Record<string, string> = {};
    for (const issue of result.error.issues) {
      const key = String(issue.path[0]);
      if (!newErrors[key]) newErrors[key] = t(issue.message, locale);
    }
    setErrors(newErrors);
    setIsSubmitting(false);
    return;
  }

  // 2. Build FormData for Formspree
  const formData = new FormData();
  formData.append('name', fields.name);
  formData.append('email', fields.email);
  formData.append('phone', fields.phone || '');
  formData.append('company', fields.company);
  formData.append('fleet_size', String(fields.fleet_size));
  formData.append('message', fields.message || '');
  formData.append('_gotcha', ''); // Honeypot — always empty; bots fill this
  // Get Turnstile token from rendered widget field
  const turnstileInput = document.querySelector<HTMLInputElement>('[name="cf-turnstile-response"]');
  if (turnstileInput?.value) formData.append('cf-turnstile-response', turnstileInput.value);
  formData.append('_next', locale === 'de' ? '/danke' : '/en/thanks');

  // 3. POST to Formspree
  try {
    const response = await fetch(
      `https://formspree.io/f/${import.meta.env.PUBLIC_FORMSPREE_CONSULT_ID}`,
      { method: 'POST', headers: { Accept: 'application/json' }, body: formData }
    );
    if (response.ok) {
      window.location.href = locale === 'de' ? '/danke' : '/en/thanks';
    } else {
      setSubmitError(t('form.error_submit', locale));
    }
  } catch {
    setSubmitError(t('form.error_submit', locale));
  } finally {
    setIsSubmitting(false);
  }
};
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser user → ConsultForm fields | Name, email, phone, company, fleet_size, message — all user-supplied strings |
| ConsultForm → Formspree API | FormData POSTed to external SaaS over HTTPS |
| Formspree → email inbox | Formspree delivers lead to Konvoi email — out of scope for this plan |
| URL params → pre-fill fields | fleet_size and estimated_savings from ROI calculator URL |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-03-01 | Injection (XSS) | ConsultForm field rendering | mitigate | Error messages sourced from translation keys only (never raw user input); field values set via React/Preact state (not innerHTML); Preact escapes all string children by default |
| T-05-03-02 | Injection (form field) | email field | mitigate | Zod validates email format (`z.string().email()`) before submission; Formspree HTML-encodes on ingest |
| T-05-03-03 | Tampering (spam) | Formspree submission | mitigate | `_gotcha` honeypot (exact field name `_gotcha`) silently rejects bot submissions; Turnstile adds client-side bot detection |
| T-05-03-04 | Denial of Service (spam flood) | Formspree endpoint | mitigate | Turnstile rate-limits bots client-side; Formspree has built-in rate limiting; server-side Turnstile Siteverify deferred to Phase 7 |
| T-05-03-05 | Information Disclosure | DSGVO consent bypass | mitigate | `z.literal(true)` rejects unchecked consent at Zod validation step — form cannot POST without consent checked |
| T-05-03-06 | Tampering (pre-fill injection) | URL params → form fields | accept | fleet_size parsed as integer (`parseInt(val, 10)`); estimated_savings parsed as integer; no HTML rendered from params; values appear in Preact-managed state only |
| T-05-03-07 | Information Disclosure | PUBLIC_FORMSPREE_CONSULT_ID in bundle | accept | PUBLIC_ prefix means intentionally client-exposed; Formspree form ID is not a secret (anyone can POST to it); spam protection is via honeypot + Turnstile, not ID secrecy |
</threat_model>
